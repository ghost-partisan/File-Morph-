##### **🛠 Project Overview**



**File Morph Pro** is built for privacy-conscious users and developers who need to process sensitive documents without cloud exposure.

This repository contains only the front-end code for the landing page.



**Key Features of the App:**



***100% Offline Processing:*** All file conversions and **OCR** happen locally on the user's hardware.



***Universal Conversion:*** Supports **HEIC, AVIF, SVG, OCR,** and various data serialization formats (**JSON, YAML, CSV**).



***Privacy First:*** Zero tracking, zero telemetry, and no data harvesting.



***Hardware Locked:*** Secure licensing tied to specific Machine IDs.
